# Autoscript Studio 🤖

> Python Automation Services for Local Businesses in Tirupati, Andhra Pradesh

A professional single-page business website for **Autoscript Studio** — a Python automation service targeting CA firms, clinics, traders, pharmacies, and schools in Tirupati, India.

---

## 📋 About This Project

AutoScript Studio helps local businesses in Tirupati eliminate repetitive manual work using custom Python automations — Excel cleanup, invoice generation, WhatsApp daily reports, data entry elimination, and more.

This website is the business's online presence — showcasing services, pricing, client types, and a contact form.

## 🚀 Getting Started

### View locally in VS Code

1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/automate-studio.git
   cd automate-studio
   ```

2. Open in VS Code:
   ```bash
   code .
   ```

3. Install the **Live Server** extension (by Ritwick Dey) from the VS Code Extensions panel (`Ctrl + Shift + X`)

4. Right-click `index.html` → **Open with Live Server**

5. Website opens at `http://127.0.0.1:5500/index.html`

### Or just open directly

Double-click `index.html` — it opens in your browser instantly. No server needed.

## 📦 Deployment

### Netlify (recommended)
1. Go to [netlify.com](https://netlify.com)
2. Drag and drop `index.html` onto the deploy zone
3. Get a live URL instantly — no account needed

### GitHub Pages
1. Rename `automate_studio.html` → `index.html`
2. Push to GitHub
3. Go to **Settings → Pages → Source → Main branch**
4. Live at `https://yourusername.github.io/automate-studio`
